# Sentiment Analysis Social Media Posts

Developed a recurrent neural network (RNN)–based sentiment analysis model leveraging Long Short-Term Memory (LSTM) units to classify social media posts into positive, negative, and neutral categories. Designed a complete NLP pipeline including text preprocessing, tokenization, and sequence padding, followed by embedding representation and sequential modelling to capture contextual dependencies.
